# Protocol Analysis

## Overview

The Protocol Hierarchy feature in Wireshark was used to identify the protocols captured during the investigation.

## Results

| Protocol | Observation |
|----------|-------------|
| IPv4 | Primary network protocol |
| TCP | Used for web communication |
| UDP | Used for DNS and NTP |
| DNS | Domain Name Resolution |
| TLS | Secure encrypted traffic |
| NTP | Time synchronization |
| ARP | Local network address resolution |

## Summary

IPv4 accounted for approximately 97.8% of all packets.

TCP represented the majority of captured traffic.

DNS accounted for approximately 2.4% of total packets.

TLS generated the largest amount of transferred bytes due to encrypted web communication.
